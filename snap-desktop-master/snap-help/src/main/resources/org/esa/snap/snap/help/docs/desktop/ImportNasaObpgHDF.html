Import NASA Ocean Color Product

The NASA Ocean Biology Processing Group (OBPG) provides harmonized ocean color data products originating from MODIS A/T, SeaWiFS, OCTS and CZCS. All data products are delivered in HDF-4 format. Please refer to the NASA for more information on the data and for data access.

The Sentinel Toolbox supports the following OBPG product types:

  • MODISA Level-2 Data
  • MODIST Level-2 Data
  • SeaWiFS Level-2 Data
  • CZCS Level-2 Data
  • OCTS Level-2 Data
