Processing Menu

The following processing operations all generate new products from one or multiple source product(s). To view their created raster datasets after processing is complete, change to the product explorer, expand the new product's nodes and double click on a band node to open an image view. Note that you really get new products. The menu contains the following groups and entries:

  • Pre-Processing
    • MERIS Cloud Probility Processor
    • MERIS Radiometric Correction
  • Thematic Water Processing
    • (A)ATSR SST Processor
    • FLH/MCI Processor
  • Thematic Land Processing
    • MERIS/(A)ATSR SMAC Atmospheric Correction
    • NDVI Processor
  • Image Analysis
    • EM Cluster Analysis
    • K-Means Cluster Analysis
    • Principle Component Analysis
    • Spectral Unmixing
  • Geometric Operations
    • Reprojection
    • Orthorectification
    • Mosaicing
    • Spatial Subset from View
    • Flipping
    • Collocation
  • Level-3 Binning
