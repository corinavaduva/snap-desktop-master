Edit Menu

The Edit Menu

Undo  

Undo the last operation

Redo  

Re-do the last operation (after Undo)

Cut    (Control-X)

Removes a selected object and, if possible, transfers it into the clipboard.

Copy    (Control-C)

Transfers the selected object into the clipboard, if possible.

Paste    (Control-V)

Transfers an object from the clipboard to the current active component, if possible.

Delete    (DEL)

Deletes a selected object. This command currently works only for the bands of a product and therefore the delete command is only enabled if a band has been selected in the product tree view or an Image View has been activated.

Select All    (Control-A)

Selects all currently selectable objects.

Deselect All    (Control-D)

Un-select all currently selected objects.

Properties...  

This menu item opens the Property Editor which can be used to edit the properties of a product, a tie-point grid or a (virtual) band.
